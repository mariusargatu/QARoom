proven caught by an AGENT-INDEPENDENT **Tier-0 falsifier**, the existing claim [`agent-test-has-teeth`](claims.md) — no new claim was added (the T25 anti-theater lesson: reuse a falsifier that already demonstrates it).


- falsifier resolves in the manifest: **yes**
- the staged cheat is real theater (a weak check greens it)

The cheat greens a weak check but the mutation gate kills it — oracle strength, not a green check, is what defends the boundary.

## What this experiment does NOT do (the deliverable is naming the limits)

1. **The full 100-feature run is the scenario this PREPARES FOR, explicitly not executed.** This is the slim scope: it wires the five metrics to their real sources and stages one caught cheat. It does not drive 100 features (a deliberate scope call).
2. **It tests THIS system, under THESE agents, against THIS threat model.** It does not prove general agent safety; the taxonomy is the measured ImpossibleBench / METR / Anthropic record, not every possible attack.
3. **The oracle problem stands** (STORY open problem 1): a maximally severe run of the WRONG property is confidently wrong, faster. These metrics measure whether the checks held, not whether the checks were the right ones.

# Falsifiable claims

> Generated by `pnpm claims:render`; do not edit by hand. 21 falsifiable claims across 8 of the 13 documented boundaries.

Don't trust the green check — flip the switch. Every claim ships with the toggle that introduces the
bug and the command that turns its gate RED: run it, and a real test fails. This page is the dare.
The live PASS/STALE verdict is per-run and deliberately NOT committed here (it would read stale the
moment the snapshot moves) — get it from `pnpm claims:verify` or a CI run's `test-results/summary.json`.
